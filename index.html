    Logic Explanation

    This program stores all loot items inside a persistent array called lootArray. Each item in the array is an object that contains a name and a value. When the user clicks the Add Loot button, the program checks the inputs using if statements to make sure the name is not empty and the value is a valid number. If the inputs are valid, the loot item is added to the array. A traditional for loop is used to go through the array and display each item in the loot list while also calculating the total value of all loot. Another loop calculates the total again when the Split Loot button is pressed and divides the amount by the number of party members. The program updates the page dynamically by creating list elements with JavaScript so the user can see the loot items and totals update in real time.
